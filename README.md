# Research and Implementation of an Identification and Monitoring System for Primary School Students on School Shuttle Buses
Hệ thống IoT giám sát học sinh trên xe đưa đón. Sử dụng STM32F4 làm Master phối hợp ESP32-S3 xử lý Edge AI nhận diện khuôn mặt. Tích hợp RFID , cảm biến ghế và module SIM 4G cảnh báo khẩn cấp. Quản lý dữ liệu qua ThingsBoard và Google Sheets.
